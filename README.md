# 風馳電掣 (Cycling Tactics)

版本: 0.1.0

## 專案概述

「風馳電掣」是一款回合制的腳踏車戰略遊戲，基於破風戰術和團隊配合。玩家需要組建三人單車隊伍，並透過巧妙的破風策略和角色輪換來擊敗對手。

## 核心玩法

- **回合制對戰**: 2位玩家輪流操作自己的隊伍
- **蛇形選角**: 平衡先後手優勢的角色選擇機制 (1-2-2-1)
- **破風戰術**: 策略性安排隊伍陣型，決定誰在前方破風
- **角色輪換**: 不同角色具有獨特的破風特性和能力
